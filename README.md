# Ubuntu-Networking-Samba-Configuration-Lab

## Project Objective

The objective of this project is to configure and manage a Linux-based system within a virtual environment, focusing on networking, user management, and secure file sharing. The lab involves setting up an Ubuntu virtual machine, configuring static IP addressing, and enabling communication between Linux and Windows systems.

Additionally, the project aims to implement and secure file sharing services using Samba, manage system users and permissions, and configure firewall rules to control access. The goal is to develop practical skills in Linux administration, network configuration, and system security within a controlled lab environment.

## MITRE ATT&CK Mapping

This project aligns with aspects of the MITRE ATT&CK framework from a defensive and system hardening perspective:

- Lateral Movement (SMB/Windows File Sharing concepts)  
- Persistence (User account management and privilege control)  
- Defense Evasion (Firewall configuration and access restrictions)  

These concepts highlight the importance of securing services, managing user access, and controlling network communication to reduce potential attack surfaces.
